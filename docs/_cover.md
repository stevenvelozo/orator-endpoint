# Orator-Endpoint

> Base endpoint class for Orator providing route definition and request handling patterns

- Declare routes in an `EndpointMethods` map, implement one handler per route
- Consistent endpoint initialization lifecycle on top of Fable
- Extended by other Orator endpoint modules
